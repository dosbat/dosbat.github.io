Python 内置函数详解

发表于2025-01-24|更新于2025-02-26|编程开发Python
|字数总计:1.8k|阅读时长:8分钟|阅读量:

前言

Python 提供了许多内置函数，这些函数可以帮助开发者更高效地编写代码。

1. abs()

返回一个数的绝对值。

2. aiter()

返回一个异步迭代器。

3. all()

如果可迭代对象中的所有元素都为真，返回 True；否则返回 False。

4. any()

如果可迭代对象中至少有一个元素为真，返回 True；否则返回 False。

5. ascii()

返回对象的可打印 ASCII 表示。

6. bin()

将整数转换为二进制字符串。

7. bool()

将一个值转换为布尔值。

8. breakpoint()

在调试时设置一个断点。

9. bytearray()

返回一个新的字节数组。

10. bytes()

返回一个新的字节对象。

11. callable()

检查对象是否可调用。

12. chr()

将整数转换为对应的字符。

13. classmethod()

将方法转换为类方法。

14. compile()

将源代码编译为代码对象。

15. complex()

创建一个复数。

16. delattr()

删除对象的属性。

17. dict()

创建一个字典。

18. dir()

返回对象的属性和方法列表。

19. divmod()

返回两个数的商和余数。

20. enumerate()

返回一个枚举对象，包含索引和相应的值。

21. eval()

执行一个字符串表达式，并返回结果。

22. exec()

执行一个字符串中的 Python 代码。

23. filter()

过滤可迭代对象中的元素。

24. float()

将一个数转换为浮点数。

25. format()

格式化字符串。

26. frozenset()

返回一个不可变的集合。

27. getattr()

返回对象的属性值。

28. globals()

返回当前全局符号表的字典。

29. hasattr()

检查对象是否具有指定的属性。

30. hash()

返回对象的哈希值。

31. help()

调用帮助系统。

32. hex()

将整数转换为十六进制字符串。

33. id()

返回对象的唯一标识符。

34. input()

从用户输入中读取一行。

35. int()

将一个数转换为整数。

36. isinstance()

检查对象是否是指定类的实例。

37. issubclass()

检查一个类是否是另一个类的子类。

38. iter()

返回可迭代对象的迭代器。

39. len()

返回对象的长度。

40. list()

创建一个列表。

41. locals()

返回当前局部符号表的字典。

42. map()

将函数应用于可迭代对象的每个元素。

43. max()

返回可迭代对象中的最大值。

44. memoryview()

返回一个内存视图对象。

45. min()

返回可迭代对象中的最小值。

46. next()

返回迭代器的下一个值。

47. object()

返回一个新的对象（通常是基类）。

48. oct()

将整数转换为八进制字符串。

49. open()

打开一个文件并返回文件对象。

50. ord()

将字符转换为对应的整数（ASCII值）。

51. pow()

返回 xx 的 yy 次方。

52. print()

输出到控制台。

53. property()

返回属性的描述符。

54. range()

返回一个可迭代的整数序列。

55. repr()

返回对象的字符串表示。

56. reversed()

返回一个反向迭代器。

57. round()

返回浮点数的四舍五入值。

58. set()

创建一个集合。

59. setattr()

设置对象的属性值。

60. slice()

返回一个切片对象。

61. sorted()

返回一个排序后的列表。

62. staticmethod()

将方法转换为静态方法。

63. str()

将对象转换为字符串。

64. sum()

返回可迭代对象中所有元素的和。

65. super()

返回父类的一个代理对象。

66. tuple()

创建一个元组。

67. type()

返回对象的类型。

68. vars()

返回对象的 __dict__ 属性。

69. zip()

将可迭代对象打包成元组。

70. __import__()

用于动态导入模块。
