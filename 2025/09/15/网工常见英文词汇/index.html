一、软件操作类

英文单词 / 词组 中文释义 补充说明
uninstall 卸载 对应 “install”（安装）的反操作，文档中提及 “uninst.exe” 为卸载程序文件，常用于软件卸载场景
install 安装 软件、程序或硬件配置的常用操作，是网工部署软件（如 VNC Enterprise Edition）的基础步骤
setup 安装 多与 “install” 搭配或单独使用，如 “setup.exe - VNC Enterprise Edition Setup”，指 VNC 企业版安装程序
accept 接受 主要用于软件安装协议确认，文档中 “accept the agreement” 是安装 VNC Enterprise Edition 等软件的前置条件，另有 “I do not accept the agreement” 选项
agreement 协议、约定 软件安装中常见 “User Agreement”（用户协议），文档内安装界面包含该选项，用户需选择接受或拒绝以继续安装流程
custom install 用户自定义安装 区别于默认安装，可自主选择需安装的组件，文档以 VNC 安装为例，可勾选 “VNC Server (64-bit)”“VNC Viewer (64-bit)” 等组件
start menu folder 开始菜单（文件夹） 软件安装时可选择是否创建，文档中有 “Don’t create a Start Menu folder” 选项，搭配 Cancel 按钮
serial 序列号 软件激活常用，文档中 “serial.txt” 为存储序列号的文本文档（340 字节）
key 密钥 用于软件授权、设备登录等，与 “serial” 功能相近，均为验证身份或权限的关键信息
license 许可 指软件使用许可，文档中有 “License.txt”（许可文档），包含软件使用权限相关条款
encryption 加密 保障数据安全的技术操作，文档中 VNC Server Properties 设置里有 “Encryption: Always”（始终加密）选项
password 密码 用户登录账户、设备或访问特定功能的验证信息，常与 “Username”（用户名）搭配使用
username 用户名 用于标识用户身份，在各类登录界面（如设备管理界面、软件登录界面）均有对应输入框
default 默认 软件安装、系统配置的预设选项，文档中 Greenshot 安装向导的 “$(default)” 代表默认组件选择
readme 说明手册 多以 “Readme.txt” 形式存在，包含软件安装步骤、功能说明、注意事项等，与 “License.txt”“Uninstall Greenshot” 等共同出现在开始菜单列表中
preference 首选项 软件个性化设置入口，文档中 Greenshot 软件的 “Preferences…” 选项，可配置捕获输出目标、插件参数等
preference options 首选项 软件或系统的个性化设置选项集合，Greenshot 的操作菜单包含 “Quick preferences”“Preferences…” 等相关选项
capture 捕获 常用于屏幕截图、网络数据包捕获，文档中 Greenshot 有 “Capture region”（捕获区域）功能，快捷键为 Ctrl+Q
region 区域 “Capture region” 中的 “region” 指截图或数据捕获的特定范围，方便精准获取所需内容
clipboard 剪切板 临时存储复制、剪切内容的空间，文档中 Greenshot 输出设置有 “Copy to clipboard” 选项，VNC Server Properties 也有 “Share clipboard with VNC Viewers” 选项
settings 设置 软件、系统的配置集合，Greenshot 的 “Settings” 包含 General、Capture、Output、Destination 等多个分类
image editor 图片（媒体文件）编辑器 用于编辑图片的工具，Greenshot 的输出目标设置有 “Open in Image editor” 选项
image 系统映像 如 “操作系统 iso” 为系统映像文件格式，文档中 “cn_windows_10_enterprise_x64_dvd_6846957.iso” 即为 64 位 Windows 10 企业版系统映像文件
shortcut 快捷方式 如 “Create a shortcut on the desktop”（在桌面创建快捷方式），软件安装时可选择创建，方便快速启动软件
register 注册 如 “register the software”（注册软件），同时 “Regedit” 为注册表编辑器，用于系统注册信息配置
Edit 编辑 如 “Edit the configuration”（编辑配置）、“Edit the file”（编辑文件），VNC Server Properties 有 “Edit…” 选项可编辑 IP 地址过滤规则
Regedit 注册表编辑器 Windows 系统工具，用于编辑注册表信息，可修改系统底层配置
version 版本 如 “software version”（软件版本）、“system version”（系统版本），文档中 Wireshark 1.4.3 (32-bit)、WinPcap 4.1.3 均为具体版本
Current 当前、目前 如 “current configuration”（当前配置）、“current version”（当前版本），“Currenty” 为拼写变体，含义相同
display 显示、查看 如 “display current-configuration”（显示当前配置）、“display IP pool”（显示 IP 地址池），是网工查看设备配置的核心命令
launch 启动、运行、加载 如 “launch the software”（启动软件）、“launch the service”（启动服务），与 “Run” 功能相近
Run 执行、运行 如 “Run the command”（执行命令）、“Run the program”（运行程序），在命令行或图形界面中启动操作的指令

二、设备与网络类

英文单词 / 词组 中文释义 补充说明
workstation 工作站 例如单个 PC，文档中 “Workstation VMware Pro” 指 VMware Pro 工作站版本，用于创建和管理虚拟机
STA （简写） 代表 PC、手机等终端设备，是网络架构中的终端节点
Enterprise 企业 常修饰软件、系统版本，如 “cn_windows_10_enterprise_x64_dvd_6846957.iso”（64 位 Windows 10 企业版镜像）
Professional 专业 代表具备专业功能的版本，区别于基础版，满足复杂网络场景下的高级功能需求
upgrade 升级 软件版本、系统功能的迭代操作，与 “update” 功能相近
update 升级 侧重系统补丁、软件小版本更新，如 “Windows Update”（Windows 更新）
basic 基本的 描述基础功能或配置模式，对应 “advanced”（高级的）
advanced 高级的 描述复杂功能或细致配置，如 “advanced options”（高级选项），VNC Server Properties 有 “Expert”（高级选项）
drive 驱动器、驱动 包含 “硬盘驱动器”（Hard-drive）、“光盘驱动器”（Cd-drive），也指设备驱动程序
disk drive 驱动器、硬盘驱动器 用于存储和读取数据的硬件，文档中对硬盘驱动器的分类和功能有明确说明
allocated 分配 指磁盘空间、内存等资源已被划分使用，与 “unallocated”（未分配）相对
unallocated 未分配 指资源未被划分使用，可对未分配磁盘空间、内存等进行规划和分配
account 账户 用户登录系统、设备的身份标识，文档中 “local-user admin”（本地管理员账户）可获取最高操作权限
secure 安全 描述安全相关的状态或设置，如 “secure connection”（安全连接）
security 安全 指安全机制、配置体系，文档中 VNC Server Properties 的 “Security” 设置可构建远程连接安全防护体系
network adapter 网络适配器、网卡 计算机连接网络的硬件，文档中 “Intel (R) 82579LM Gigabit Network Connection” 为具体网卡型号
Gigabit 吉比特、千兆 用于描述网络速率，“1000Mbps” 即千兆比特每秒，“GigabitEthernet” 为千兆以太网接口
Ethernet 以太网 主流局域网技术，“Fast Ethernet (FE)”（快速以太网）是其分支，速率通常为 100Mbps
Fast Ethernet(FE) 快速以太网 以太网的子类型，速率 100Mbps，接口标识如 “E0/0/2”（E=ethernet）
command 命令、指令 命令行模式下的操作指令，是网工通过 CLI 配置路由器、交换机等设备的核心手段
Cmd （简写） 即 “Command”，指命令行程序，是进入命令行模式、执行配置命令的入口工具
CLI command line（命令行模式） 设备配置的重要方式，通过输入命令实现精准配置，是常用配置方式（CLI、Web、客户端）之一
Web 网页、图形界面 设备的图形化配置方式，无需记忆复杂命令，适合快速完成基础配置
Client 客户端 与 “server”（服务端）相对，如 “VNC Client”（VNC 客户端），用于访问服务端提供的服务
server 服务端 提供网络服务的设备或程序，如 “VNC Server”（VNC 服务端），是远程控制、数据共享等服务的核心提供方
config 配置 “configure” 和 “configuration” 的简写，如 “config the interface”（配置接口）
configure 配置 对设备、软件进行参数设置的操作，如 “configure the router”（配置路由器）
configuration 配置 指配置的过程或结果，“display current-configuration”（显示当前配置）可查看设备当前配置状态
remote desktop 远程桌面 简写 “RDP”，用于远程控制计算机桌面，是异地管理设备、排查故障的重要工具
Remote 对端、远端 与 “local”（本地、本端）相对，如 “remote server”（远端服务器）
local 本地、本端 如 “local disk”（本地磁盘）、“local configuration”（本地配置）
Guest 来宾 指权限受限的账户，文档中 VNC Server Properties 有 “Guest login access”（来宾登录权限）设置
Administrator 管理员 拥有最高操作权限的账户，可执行设备的所有配置和管理操作
defender 防御 如 “Windows Defender”（Windows 防御程序），可查找恶意软件并阻止其运行
WinPE Windows Preinstallation Environment（Windows 预安装环境） 系统维护、安装的专用环境，“电脑店”“大白菜”“U 大师” 等均为基于 WinPE 的工具
boot 启动 系统、设备的启动过程，如 “boot from local drive”（从本地驱动器启动）
startup 启动 与 “boot” 相近，如 “startup options”（启动选项），可配置设备启动模式（UEFI / 传统启动）
Memory 内存 计算机的临时存储设备，如 “system memory”（系统内存），“kdump” 需预留部分系统内存
OS Operating System（操作系统） 如 Windows、Linux，文档中涉及 Windows 10、Red Hat Enterprise Linux 等
cache 缓存 用于临时存储数据以提升访问速度，如 “disk cache”（磁盘缓存）、“network cache”（网络缓存）
I/O input、out（输入、输出） 包括 “网络 IO”“硬盘 IO”，指数据的输入输出操作，是分析设备数据处理能力的重要维度
device 设备 如 “network device”（网络设备：路由器、交换机）、“storage device”（存储设备：硬盘、U 盘）
chipset 芯片 设备中的核心芯片组，如 “motherboard chipset”（主板芯片组），影响硬件兼容性和性能
Secure Boot 安全启动 保障启动安全的机制，“Secure Boot Mode” 有 Disabled、User Mode、Standard Mode 等选项
Operating Mode 操作模式 如 “Secure Boot Operating Mode”（安全启动操作模式），不同模式对应不同功能状态
disable 禁用、停止、关闭 与 “enable”（启用）相反，如 “disable the interface”（禁用接口）
enable 启用、使能、开启 如 “enable the network adapter”（启用网络适配器），保障设备正常运行
Cd-drive 光盘驱动器 用于读取光盘的设备，在安装传统光盘版系统、软件时使用
Hard-drive 硬盘、硬盘驱动器 计算机的主要存储设备，文档中对硬盘分区（如 System (C:)、Network (D:)）、空间大小有详细描述
Removable 可移动设备 如 U 盘，“Removable disk”（可移动磁盘），方便携带数据、临时部署工具
Disk 磁盘 包括硬盘、软盘等，如 “local disk”（本地磁盘）、“floppy disk”（软盘）
Floppy disk 软盘 早期可移动存储设备，当前使用较少，是了解存储设备发展历程的重要概念
discard 丢弃、遗弃 如 “Discard Changes”（丢弃更改），PhoenixBIOS Setup Utility 有该选项，可放弃无效配置
save 保存 保存配置、文件等，如 “Save Changes”（保存更改），确保配置不丢失
exit 退出 退出程序、配置界面，如 “Exit Saving Changes”（保存更改后退出）
load 加载、恢复 如 “load setup default”（恢复出厂设置），可加载默认配置恢复设备初始状态
load setup default 恢复出厂设置 将设备配置恢复到初始默认状态，配置混乱或故障时可重置设备
partition 分区 对磁盘进行逻辑划分，如文档中 “System (C:)”“workroom (E:)” 等分区，方便分类存储数据
cancel 取消 取消当前操作，如 “Cancel the installation”（取消安装），可终止错误或不必要的流程
commit 确认、确认生效、提交生效 确认配置并使其生效，如 “commit the changes”（提交更改），确保配置作用于设备
Legacy 传统 如 “UEFI/Legacy Boot”（UEFI / 传统启动），指传统的 BIOS 启动方式，与 UEFI 启动相对
Compatibility 兼容性 如 “Compatibility Mode”（兼容模式），保障软件或设备在不同环境下正常运行
Module 模块 软件或硬件的功能单元，如 “network module”（网络模块）、“plugin module”（插件模块）
slot 插槽、板块 硬件设备的接口插槽，如 “PCI slot”（PCI 插槽）、“network card slot”（网卡插槽）
power 电源、能量 如 “power supply”（电源供应器）、“power button”（电源按钮），是设备正常运行的能源保障
Shadow 影子 如 “影子系统 PowerShadow”，文档中提及 PowerShadow8.5 的安装流程，可创建虚拟环境保护系统
reboot 重启 重新启动设备，如 “reboot the system”（重启系统），解决系统小故障、使配置生效
reload 重启、重新加载 重新加载配置或程序，如 “reload the configuration”（重新加载配置）
restart 重启 与 “reboot” 相近，如 “restart the service”（重启服务），保障网络服务正常运行
info 信息、提示 “information” 的简写，如 “system info”（系统信息），快速获取设备关键状态信息
information 信息、提示 如 “error information”（错误信息），帮助排查故障、了解设备运行异常原因
keyboard 键盘 计算机输入设备，VNC Server Properties 有 “Enable keyboard input”（启用键盘输入）设置
Mouse 鼠标 计算机输入设备，VNC Server Properties 有 “Enable mouse input”（启用鼠标输入）设置
Files 文件 如 “file transfer”（文件传输），VNC 软件支持 “Share files with VNC Viewers”（与 VNC 查看器共享文件）
Chat 聊天 如 “Enable chat”（启用聊天功能），VNC 软件支持该功能，可实现远程沟通
Connected 连接 描述设备或网络处于连通状态，如 “connected to the network”（连接到网络）
Print 打印 如 “Desktop Printing”（桌面打印），VNC Server Properties 有该功能设置，支持远程打印
Expert 高级选项 如 “Expert Settings”（高级选项设置），VNC Server Properties 中的 “Expert” 标签页提供专业配置项
Port 端口号 网络服务的标识编号，如 “configure port numbers”（配置端口号），保障服务正常区分和访问
source 源 简写 “src”，如 “source IP address”（源 IP 地址），定位网络数据发送源头
src 源 “source” 的简写，在路由配置、流量过滤规则中频繁使用
destination 目标 简写 “dst”，如 “destination IP address”（目标 IP 地址），确定网络数据接收方
dst 目标 “destination” 的简写，是配置路由、防火墙规则的核心参数
Filter 过滤 如 “IP Address Filtering”（IP 地址过滤）、“traffic filter”（流量过滤），VNC 软件支持 IP 地址过滤
traffic 流量、数据流量 如 “network traffic”（网络流量），通过分析流量了解网络负载、排查异常传输
Reject 拒绝 与 “deny”“permit” 配合使用，如 “reject the invalid connection”（拒绝无效连接）
deny 拒绝 如 “deny the unauthorized IP”（拒绝未授权 IP），在防火墙规则、访问控制列表中常用
permit 允许 如 “permit the trusted traffic”（允许可信流量），保障合法数据正常传输
Disconnect 断开连接 如 “disconnect the idle session”（断开空闲会话），释放网络资源
Auto-xxx 自动 “xxx” 代表具体功能，如 “Auto-save”（自动保存）、“Auto-connect”（自动连接），简化操作
Authentication 认证 验证用户或设备身份的过程，如 “Authentication scheme”（认证方案），VNC 软件支持多种认证方式
scheme 方案 如 “authentication scheme”（认证方案）、“route-policy scheme”（路由策略方案），是配置复杂功能的模板
Status 状态 描述设备、服务的运行情况，如 “connection status”（连接状态）、“service status”（服务状态）
State 状态 与 “Status” 相近，如 “system state”（系统状态）、“interface state”（接口状态）
Login 登陆 用户进入系统的操作，如 “Guest login access”（来宾登录权限），控制用户登录权限范围
Access 接入、进入 如 “access control”（访问控制）、“network access”（网络接入），保障资源不被未授权访问
Screen 屏幕 如 “blank the screen”（黑屏）、“screen capture”（屏幕捕获），VNC 和 Greenshot 均有屏幕相关功能
Desktop 桌面 如 “desktop background”（桌面背景）、“remote desktop”（远程桌面），是远程控制、个性化配置的核心对象
Optimisation 优化 如 “screen capture optimisation”（屏幕捕获优化），Greenshot 软件支持该功能提升截图效率
Privacy 隐私 如 “privacy settings”（隐私设置），VNC 软件的 “Blank the screen while VNC Viewers are connected” 功能保护桌面隐私
extend 扩展、额外 如 “extend the disk space”（扩展磁盘空间）、“extend the network”（扩展网络），满足扩容需求
cipher 加密 与 “encryption” 相近，如 “encryption cipher”（加密算法），文档中 “local-user admin password cipher” 表示密码加密存储
none 空、空白、没有、无 如 “Encryption: None”（不加密），测试环境中可选择该模式简化操作
blank 空白、清空 如 “blank the screen”（清空屏幕），VNC 软件的隐私保护功能
copper 铜芯线、双绞线 用电传输的网线，华为设备 “combo-port copper”（组合接口铜缆模式），适用于短距离连接
Fiber 光纤 高速数据传输介质，华为设备 “combo-port fiber”（组合接口光纤模式），适用于长距离、高速率传输
Auto 自动 如 “combo-port auto”（组合接口自动模式），设备可自动识别连接介质（铜缆 / 光纤）
Serial 串行 如 “Serial interface”（串行接口）、“serial cable”（串行电缆），常用于设备 console 口配置
POS （特定接口类型） 常用于特定设备（如 POS 机）的连接，是特殊设备接口类型
Console 控制台 设备配置接口，如 “Console port”（控制台端口），通过 console 线连接可初始化或调试设备
Combo 混合接口 如 “combo-port”（组合接口），支持铜缆和光纤两种连接方式，华为 S3700 设备有该接口
